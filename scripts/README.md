# Scripts

This directory contains utility scripts for the Horde Model Reference project.

## Available Scripts

### `sync_github_references.py`

Syncs model references from a PRIMARY server to GitHub legacy repositories.

**See:** [SYNC_README.md](../SYNC_README.md) for detailed documentation.

### `get_all_names.py`

Utility script to extract all model names from the model reference files.

### `create_example_json.py`

Generates example JSON files for testing and documentation purposes.
